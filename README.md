# Telangana Investment Analysis 2019-2023

## Project Overview
Analysis of 28,770 investment records from Telangana TS-iPASS government data 
to identify investment patterns, top districts and sector performance.

## Tools Used
- Excel — Data exploration and VLOOKUP
- SQL (SQLite/DBeaver) — Business queries
- Power BI — Interactive dashboard

## Key Findings
1. Rangareddy district leads with ₹42,706 Crores investment
2. Wood & Leather sector creates 23 jobs per ₹1 Crore — most efficient
3. Investment peaked in 2020 at ₹31,191 Cr then dropped due to COVID
4. Real Estate dominates at 30% of total investment
5. Mulugu is most underserved district with only ₹34 Crores
